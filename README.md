# Overview

This repository contains the backend services for the Digital Life Lessons platform. It provides secure REST APIs for authentication, lesson management, premium subscriptions, favorites, comments, reports, and administrative operations.

The server is built with **Node.js**, **Express.js**, **MongoDB**, **Better Auth**, and **Stripe** while following secure RESTful API development practices.
